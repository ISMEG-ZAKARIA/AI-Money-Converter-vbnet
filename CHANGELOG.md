## [1.0.0] - 2026-03-19

### Added
- Initial public release of the AI Money Converter.
- Integrated OpenAI GPT-3.5 Turbo for real-time currency conversion calculations.
- Integrated OpenAI GPT-4 Turbo for the conversational AI Chatbot assistant.
- VB.NET Desktop Application built with .NET 8.0 WinForms.
- Multi-currency support for common global currencies.
- Professional project structure with security and contribution policies.
- Clean and intuitive user interface for currency conversion and assistant interaction.
