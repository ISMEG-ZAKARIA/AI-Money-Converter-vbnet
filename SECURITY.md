# Security Policy

## Supported Versions

The following versions of this project are currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

As a cybersecurity professional, I take security seriously. If you discover a security vulnerability in this project, please follow these steps:

1. **Do not disclose the vulnerability publicly.** This includes opening a GitHub issue or posting on social media.
2. Please report the vulnerability by contacting the author directly via GitHub.
3. Provide a detailed description of the vulnerability, including steps to reproduce it.
4. I will acknowledge receipt of your report and provide a timeline for addressing the issue.

Thank you for helping to keep this project secure!

### Responsible Disclosure

- Allow me a reasonable amount of time to fix the issue before disclosure.
- Do not exploit the vulnerability further than necessary to prove its existence.
- I will credit you for the discovery in the CHANGELOG unless you prefer to remain anonymous.
